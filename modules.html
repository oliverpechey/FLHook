Plugins
Here is a list of all modules:
 Away from KeyboardThe AFK plugin allows you to set yourself as Away from Keyboard. This will notify other players if they try and speak to you, that you are not at your desk
 Anti Jump DisconnectThe "Anti Jump Disconnect" plugin will kill a player if they disconnect during the jump animation. If tempban is loaded then they will also be banned for 5 minutes
 ArenaThis plugin is used to beam players to/from an arena system for the purpose of pvp
 AutobuyThe "Autobuy" plugin allows players to set up automatic purchases of various munition/consumable type items
 BettingA plugin that allows players to place bets and then duel, the winner getting the pot
 Bounty HuntThe "Bounty Hunt" plugin allows players to put bounties on each other that can be collected by destroying that player
 Cargo DropThe "Cargo Drop" plugin handles consequences for a player who disconnects whilst in space and allows servers to ensure that cargo items are dropped on player death rather than lost. Is also allows server owners to set additional commodities that are spawned as loot when a player ship dies based on hull mass (i.e. salvage)
 Cash ManagerThe "Cash Manager" plugin serves as a utility for which players can manage their money through a shared bank and transfer money between characters with ease. It also exposes functionality for other plugins to leverage this shared pool of money
 Connection DataThis plugin is used to provide connection data e.g. ping
 Crash CatcherThis plugin is used to catch and handle known crashes in FLServer
 Death PenaltyThis plugin charges players credits for dying based on their ship worth. If the killer was a player it also rewards them
 EventThis plugin is used to create NPC and Cargo missions. For example, a base needs to be sold a certain amount of a commodity
 IP BanThis plugin is used to ban players based on their IP address
 Kill TrackerThis plugin is used to count pvp kills and save them in the player file. Vanilla doesn't do this by default. Also keeps track of damage taken between players, prints greatest damage contributor
 Light ControlAdds functionality to change the lights on player ships
 Loot TablesThis plugin implements functionality which allows for more control over what items get dropped when a ship is destroyed, and over their drop probabilities
 MarkA plugin that allows players to mark objects for themselves or group
 MessageHandles different types of messages, banners and faction invitations
 Mining ControlAdds bonuses to mining
 Misc CommandsThe "Misc Commands" plugin provides an array of useful commands that serve as general quality of life features or game enhancements. They are commands that do not really fit into another plugin, or have the significance to belong in FLHook Core
 NPC ControlThe NPC Control plugin allows admins to spawn and control NPC ships using any ship loadout in the server files. This is especially useful during events
 StatsThe plugin tries to normalize the multiplayer specific unnatural physics
 Purchase RestrictionsThe Purchase Restrictions plugin restricts the purchase of equipment, goods and ships unless the player holds a certain item
 RenameThe plugin allows players to rename their characters, move them between accounts and create password-protected player tags
 Sound ManagerThe plugin plays a random sound upon player login. To be expanded upon
 System SensorThe plugin allows players with proper equipment to see player traffic coming through Trade Lanes and Jump Gates in the system, as well as being able to look up their equipment and cargo at the time of using them
 TaxThe Tax plugin allows players to issue 'formally' make credit demands and declare hostilities
 WardrobeThe Wardrobe plugin allows players to change their body and head models from a defined list of allowed models
 WarehouseThe Warehouse plugin allows players to deposit and withdraw various commodities and equipment on NPC stations, stored on internal SQL database
