Kill Tracker

Detailed Description

This plugin is used to count pvp kills and save them in the player file. Vanilla doesn't do this by default. Also keeps track of damage taken between players, prints greatest damage contributor.

Player Commands

All commands are prefixed with '/' unless explicitly specified.

  • kills {client} - Shows the pvp kills for a player if a client id is specified, or if not, the player who typed it.

Admin Commands

There are no admin commands in this plugin.

No configuration file is needed.
